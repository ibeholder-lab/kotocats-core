# kotocats-core

Общий модуль кошек для проектов:

- сайт фонда
- сайт котокафе
- бот котокафе
- kotopay при необходимости

## Что внутри

- загрузка кошек из Directus
- нормализация данных кошки
- медиа кошек
- кеширование каталога
- роуты `/cats` и `/cats/:slug`
- общие EJS partials
- общий JS доната

## Подключение в проект

```bash
npm install ../kotocats-core
